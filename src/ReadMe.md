### @Intro
* 用电客户流入流出情况分析  
    对该模块的分析，_房屋空置率是研究的重点_，可以从单个家庭用户的电能数据入手来作为判断房屋空置的依据. 
* 用电客户用电规划及建议  
    对用电用户在不同季节，不同时间段，以及天气影响等多维度展开分析及数据挖掘工作，统计用户在不同的时间内的用电量不同。预测用户下一段的用电量，根据预测用电量，为用电用户提供相应的用电规划和用电建议。
